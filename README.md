# 🍷 Sistema de Inventario - La Costera 28

## 📖 Descripción del Proyecto
Este proyecto es un Prototipo Funcional de un sistema web integral con arquitectura cliente-servidor orientado a servicios, desarrollado para el restaurante "La Costera 28" (Bacalar). Su objetivo es digitalizar la gestión de la cava de vinos, permitiendo registrar entradas, salidas, alertas de stock mínimo y una administración segura mediante autenticación de usuarios.

## 🛠️ Tecnologías Utilizadas

**Backend (API REST):**
* Node.js y Express (Servidor web)
* PostgreSQL (Base de datos relacional)
* JSON Web Tokens (JWT) (Autenticación y seguridad de rutas)
* bcrypt (Encriptación de contraseñas)
* CORS y dotenv

**Frontend (Cliente):**
* HTML5 y CSS3
* JavaScript Vanilla (Consumo de API mediante Fetch)
* Bootstrap 5 (Diseño responsivo, minimalista y funcional)
